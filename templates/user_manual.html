Further Informations

1. Uploading event logs

Please click the button "Manage" on the menu bar to get to the page where you are able to upload a new event log.





After you are on the Manage page of the tool, please click the button "Choose file".





Next, choose a .xes/.csv file you want to upload (1.). Afterwards, please click the button "Open" (2.).





After the .xes/.csv file is chosen, the last step is to click the button "Upload EventLog" to upload it.





If your upload was successful, you will see the following "Upload successful" message.





2. Deleting event logs

Please click the uploaded event log you would like to delete (1.). Thereafter, click the button "Delete" (2.) to erase the selected event log.





If your deletion was successful, you will see the following "Successfully deleted" message.





3. Choose Number of Comparisons

Please click the button "Compare" on the menu bar to get to the page where you are able to choose the number of comparisons.





After you are on the "Compare" page, please select the number of comparison you want to make between 2 and 8 (1.)
and confirm your choice by clicking the button "Continue".





4. Select Uploaded Event Logs

After you have confirmed your selected number of comparisons, please now select the uploaded event logs you want to compare
by clicking the drop-down lists.





You can choose an event log by clicking each of the drop-down lists.





After you have completed your selection, click the button "Continue" to confirm it.





5. Pick Reference Log

As a default implementation the first (far left) log is the reference log. The reference log can be recognized by the green font of the event log title.
If you click the title of another event log, this log will be selected as the new reference log.





Consequently, the new reference log can be recognized by the green writing of its title.





6. Select Filter

6.1. Choose Edge Labeling:
Click on the first (top) filter to select the edge label.





For the graphical depiction of the edge in the Directly-Follows Graph (DFG), you can select either "Frequency" or "Performance".





In addition, if "Performance" is chosen, there is a further option to select between different time units. To do this, click on the list with the default label seconds.





You can select between a visualization in:

  • "Seconds"
  • "Minutes"
  • "Hours"
  • "Days"





As an example you can see the selection "Days".




6.2. Selected Filter Type:
To select a filter, click on the "Select Filter" listto select the edge label.





The selectable filters are:

  • "Case Performance"
  • "Between Filter"
  • "Case Size"
  • "Timestamp (Contained)"
  • "Timestamp (Intersecting)"
  • "Filter on Attributes"





6.3. Two Example Cases to demonstrate the selection of the filters:

  • 6.3.1. Between Filter
  • 6.3.2. Filter on Attributes


6.3.1. Between filter:
Regarding the Between filter, click on two list below the "Between Filter" selection in the "Select Filter" list.
As a default choice you will see one activity of the Event Log (here: "Create Fine").





Next, choose the first and second activity. The first one will be the start activity and the second one the end activity.





Lastly, click on "Apply Filter" to apply the selected Between filters.





In this case, the following Directly-Follows Graph is the result.





6.3.2. Filter on Attributes:
For each type of Attribute Filter, there are several attributes to choose from or options to type in a value for filtering.
First of all, click on the attributes list to select an Attribute Filter type. As a default choice you will see one attribute of the Event Log (here: "amount").





Next, choose the attribute from the list you would like to use for filtering.





Then select the parameters that correspond to the filtering you want. For this specific type of attribute filtering, click on the associated list.





Next, select a value from the list.





Then click on the associated operator list.





And select an operator that correspond to the filtering you want.





Lastly, click on "Apply Filter" to apply the selected Attribute Filter.





In this case, the following Directly-Follows Graph is the result.





7. Reset Filter

Please click on the "Reset Filter" button to reset the applied filters.





Afterwards, the following Directly-Follows Graph without any applied filters is the result.





8. Directly-Follows Graph (DFG)

The Directly-Follows Graphs (DFGs) show all the event logs based on the previously selected filters.
In addition, all displayed DFGs are compared to the reference log.
If nodes are part of the reference log, they are highlighted in blue. If nodes are not part of the reference log, they are highlighted in red.





9. Metrics

The metrics of all displayed event logs are shown. In addition, if an event log is different to the reference log, the difference between the logs will be displayed, too.
(A filtered log that is identical to the reference log, but with chosen filters that lead to a different DFG, will be counted as a different log as well.)





10. PDF Download

After the appropriate event logs and filtes have been selected as well as the representation of the DFGs and metrics made,
it is now possible to export the gathered information in PDF format by clicking the "Download" button.





11. New Comparison

Either click "Compare" or "Manage" on the bar menu.
"Compare" is the correct option, if you don't want to upload an additional event log prior to the new comparison.





But, if you want to upload a new event log, please go to the "Manage" page first and upload it.





In accordance, either move on to 1. Uploading event logs or 3. Choose Number of Comparisons in the User Manual to continue.
