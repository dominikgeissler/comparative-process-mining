Comparative Process Mining

User Manual

1. Uploading event logs


Please click on the button "Manage" on the menu bar to get to the page where you are able to upload a new event log.




After you are on the "Manage" page, please click on the button "Choose file".




Now, you can choose a .xes/.csv file.
Please click on the .xes/.csv file you want to upload (1.) and afterwards click on the button "Open" (2.).




After the .xes/.csv file is choosen and the last step is to click on the button "Upload EventLog" to upload it.




If your upload was successful, you will see the following "Upload successful" message.



2. Deleting event logs


Please pick the uploaded event log you would like to delete by clicking on it (1.).
Thereafter, click on the button "Delete" (2.) to erase the selected event log.




If your deletion was successful, you will see the following "Successfully deleted" message.



3. Choose Number of Comparisons

Please click on the button "Compare" on the menu bar to get to the page where you are able to choose the number of comparisons.




After you are on the "Compare" page, please select the number of comparison you want to make between 2 and 8 (1.)
and confirm your choice by clicking on the button "Continue".



4. Select Uploaded Event Logs

After you have confirmed your selected number of comparisons, please now select the uploaded event logs you want to compare
by clicking on the drop-down lists.




By clicking on each of the drop-down lists, you can choose an event log.




After you have completed you selection, you confirm it by clicking on the button "Continue".



5. Pick Reference Log

As a default implementation the first log is the reference log.
The reference log can be recognized by the green font of the event log title.
By clicking on the title of another event log, this log will be selected as the new reference log.




Consequently, the new reference log can be recognized by the green writing of its title.

6. Select Filter

1. Choose Edge Labeling:
For the graphical depiction of the edge in the Directly-Follows Graph (DFG), you can select either "Frequency" or "Performance".






In addition, if "Performance" is chosen as edge labeling,
there will be an further option to select between a visualization in "Seconds", "Minutes", "Hours" or "Days".







2. Selected Filter Type:
The filters "Case Performance", "Between Filter", "Case Size", "Timestamp (Contained)", "Timestamp (Intersecting)" or "Filter on attributes" can be selected.

8. Directly-Follows Graph (DFG)

First of all, Directly-Follows Graphs (DFGs) show all the event logs based on the previously selected filters.
According to this fact, the edges for example will be either labeled with "Frequency" or "Performance" values.
Plus, in the case of "Performance" labels either in "Seconds", "Minutes", "Hours" or "Days".
Second, the DFGs display all uploaded event logs and compare them with the picked reference log.
Nodes that are similar to the reference log, will be highlighted in blue.
On the other hand, the nodes that are different to the reference log, will be highlighted in red.

9. Metrics

The metrics of all event logs will be displayed. In addition, for all event logs that are different to the reference log, the difference between the logs will be displayed.
Hereby, a filtered log that is the same log as the event log, but with choosen filters that lead to a different DFG,
will be counted as a different log.

10. PDF Download

After the appropriate event logs and filtes have been selected as well as the representation of the DFGs and metrics made,
it is now possible to export the gathered information in PDF format by clicking on the "Download" button.

11. New Comparison

Either click on "Compare" or on "Manage" on the bar menu.
"Compare" is the correct option, if you don't want to upload an additional event log prior to the new comparison.




But, if you want to upload a new event log, please go to the "Manage" page first and upload it.
In accordance, either move on to 1. Uploading event logs or 3. Choose Number of Comparisons in the User Manual to continue.
